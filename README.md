# 💅 BeautyCRM

A simple CRM system for beauty salons built with **Python**, **Flask**, and **SQLite**.

## ✨ Features

- ✅ Add new clients
- ✅ View all clients
- ✅ Edit client information
- ✅ Delete clients
- ✅ Add comments to appointments
- ✅ User-friendly Bootstrap interface

## 🛠 Technologies

- Python 3
- Flask
- SQLite
- HTML5
- Bootstrap 5

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/BeautyCRM.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the database script:

```bash
python database.py
```

Start the application:

```bash
python app.py
```

Open your browser:

```
http://127.0.0.1:5000
```
